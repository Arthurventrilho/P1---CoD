Descrição do Projeto:

O foco deste projeto é a modelagem de sistemas mecânicos – “mecânicos” no sentido comumente utilizado na Física, ou seja, o estudo do movimento de corpos macroscópicos. Este modelo será composto de sistemas de equações diferenciais, obtidas a partir do uso de diagramas de corpo livre.

Conhecimentos Utilizados:

Objetivos de Aprendizagem:

  1. Criar modelos de diferentes tipos de sistemas, usando diferentes tipos de abstrações
  2. Implementar solução numérica de modelos de sistemas dinâmicas usando a linguagem de programação Python
  3. Utilizar as técnicas mais comuns de validação de modelos a partir de dados dos sistemas reais
  4. Elaborar conclusões a partir do comportamento observado para os modelos simulados
  5. Comunicar oralmente, com auxilio de um pôster ou slides, conteudo técnico
